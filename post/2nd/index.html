How to rebuild my hugo file

1) Delete ‘public’ folder Link to heading

if ‘Already exists in the index~’ error occured, should delete caches in your directory.

open git bash!

git rm --cached [my hugo project folder]

2) Open ‘git bash’ Link to heading

Link to heading

3) Go to my hugo project folder Link to heading

use “cd” command

Link to heading

4) link with my repository Link to heading 

git init
git remote add origin <my github repository(for build) url>

5) my github.io respository should be submodule. Link to heading 

git submodule add -b master <user.github.io repository url> public

6) Let’s build Link to heading 

hugo -t <theme name>

then, you can see the files in public folder in your project directory

7) push files Link to heading 

$ cd public
$ git add .
$ git commit -m "<commit message>"
$ git push origin master
$ cd ..
$ git add .
$ git commit -m "<commit message>"
$ git push origin master
